RF.py in code directory is the main script. The model is the working directory, where predictors/total.csv is the data file.

Predictors:
crop - cropland types
rain - annual precipitation
irrigation - irrigated area density
salt - salt water distribution
distance - canal distance
pop - population density
AI - aridity index
building - building density
CGI - cropping intensity
clay - clay content
soim_m - soil moisture
tem - mean temperature

DGWS - classified groundwater storage change rates in confined aquifers
